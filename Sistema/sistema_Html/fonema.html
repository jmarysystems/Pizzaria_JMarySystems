FONEMA
DEFINIÇÕES

Fonema é a menor unidade de som capaz de fazer distinção entre duas palavras.

1) Fonema é a menor unidade sonora do sistema fonológico de uma língua. Fonologia é a disciplina que estuda cada um dos sons da voz.

2) É o som da fala.

3) Fonema é a unidade mínima sonora que é capaz de estabelecer diferenciação entre um vocábulo e outro.

4) Fonemas são as entidades capazes de estabelecer distinção entre as palavras. Exemplos: casa/capa, muro/mudo, dia/tia.
